# seat-wind-assigner
1. 制作物の概要、特徴、機能
   - 麻雀の席決めを行う際に使うギミック。
   - 裏向きの牌にインタラクトすると表向きになり、東南西北のいずれかが分かるようになる。
   - シャッフルボタンを押すと牌が裏向きになり、シャッフルされる。

2. 開発の背景  
   - 独自で作成したもの。
   - 席決めをより現実に近い形で行い、没入感を高めるため。

3. 制作物の構成、アーキテクチャ  
   - Unity、UdonSharp（VRChat向けに開発されたC#のコンパイラー）。

4. 開発期間・時間  
   - 約1週間。

5. 工夫した点・課題点等  
   - 牌をめくる際に音が鳴るようにした。  
   - 将来的に1つのプログラムにまとめたい。Unityの仕様上、Interactメソッドの内容を使い分ける場合はスクリプトファイルが2つ以上必要になる。そのため、シャッフルボタンをメソッドの指定が可能なUIボタン方式に置き換え、Interactメソッドを使わないシステムに変更したい。
